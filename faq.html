❓ 常見問題

解答您關於活動的各種疑問

本次迎新宿營開放給台灣大學資訊管理學系、心理學系、物理治療學系的112學年度入學新生參加。如果您是這三個系所的新生，都歡迎報名參加！

報名費用為 NT$ 2,800，包含以下項目：

  • • 三天兩夜住宿費用
  • • 六餐餐費（第一天午晚餐、第二天三餐、第三天早餐）
  • • 活動材料費與場地費
  • • 營隊紀念品
  • • 交通接駁費用
  • • 保險費用

請依照以下步驟完成報名：

  1. 1. 填寫線上報名表
  2. 2. 繳交報名費（銀行轉帳或現金繳費）
  3. 3. 上傳家長同意書（未滿20歲需要）
  4. 4. 上傳身分證影本
  5. 5. 等待報名確認通知

報名截止日期為 2025年7月15日。名額有限，建議儘早報名以免向隅！如果報名人數超過預期，將採先報名先錄取的方式。

我們的活動設計考量到不同個性和體能狀況的同學。大部分活動都是團體合作性質，不會有過於激烈的運動。如果您有特殊身體狀況或需求，請在報名時告知我們，我們會協助安排適合的替代活動。

我們已準備完整的備案計畫。如遇下雨，戶外活動將改為室內進行；如遇颱風等惡劣天候，營隊可能延期或改為一日活動。所有變更都會提前通知參加者，並依情況退還部分費用。

必備物品：

  • • 身分證件、健保卡
  • • 個人藥品（如有需要）
  • • 換洗衣物（建議多帶一套）
  • • 盥洗用具
  • • 毛巾、拖鞋

建議攜帶：

  • • 防曬用品、帽子
  • • 雨具
  • • 相機或手機
  • • 小零食
  • • 個人水壺

我們入住的是合歡自然生活體驗基地的多人宿舍，每間房間約4-6人，備有基本寢具。浴室為共用，男女分開。基地環境優美且安全，並有專人管理。建議自備毛巾和個人盥洗用品。

我們提供兩種交通方式：

統一接駁：8月24日早上8:00在台大校門口集合，搭乘遊覽車前往

自行前往：請於當日上午10:00前抵達基地，詳細路線圖可在下載區取得

建議選擇統一接駁，既安全又能與同學提早認識！

基地提供免費WiFi，訊號穩定。每間宿舍都有充電插座，但數量有限，建議攜帶延長線或行動電源。不過，我們鼓勵大家多與夥伴面對面互動，適度使用3C產品！

請在報名時詳細填寫飲食需求（如素食、過敏原等），我們會與基地廚房協調準備合適的餐點。一般素食需求都能配合，特殊過敏原請提早告知以便安排。

退費標準如下：

  • • 活動前14天取消：退還90%費用
  • • 活動前7天取消：退還70%費用
  • • 活動前3天取消：退還50%費用
  • • 活動前1天或當日取消：不予退費

如因不可抗力因素（如疫情、天災）需取消，將全額退費。

  • • 活動諮詢專線：(02) 3366-1234
  • • 辦公時間：週一至週五 9:00-17:00

我們會盡快回覆您的疑問！
